RainNest is an IoT powered umbrella sharing system that helps people stay prepared for unexpected rain. Using smart kiosks, mobile apps, and cloud connectivity, users can easily borrow and return umbrellas anytime.

About

What is RainNest?

RainNest is a revolutionary urban mobility solution designed to provide weather protection on demand. Our network of smart IoT kiosks ensures you're never caught off guard by the rain. By integrating sensors, mobile apps, and real-time cloud data, we're building infrastructure for a smarter, more resilient city.

How It Works

Simple Steps

  • 01 Open the RainNest mobile app
  • 02 Locate a nearby kiosk on the map
  • 03 Scan the QR code at the kiosk
  • 04 Take the umbrella
  • 05 Return it to any kiosk when done
